    JS的展开语法(...)

    kkyeer 11/24/2018 JS

    # JS的展开语法(...)

    1.rest操作符，用在函数参数列表里，结构参数列表...args function funcName{ /* 此时args等同于arguments，参数名args可随意定义 */ } 示例：

    function test(...params){
    console.log(params);
}
test('ha','hei')
// [""ha"", ""hei""]
进阶用法：
function func2(param1,...param2){
    console.log(param1);
    console.log(param2);
}
func2('p1','p2')
// p1
// [""p2""]

console.log(arr1)
// str1
